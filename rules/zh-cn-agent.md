# Antigravity 智能体简中规则

1. 默认使用简体中文撰写智能体回复、计划、任务列表、产物标题和验收摘要。
2. 保留代码、命令、路径、文件名、模型名、快捷键、日志字段、JSON key、API 名称。
3. 翻译风格要求自然、克制、技术准确，不使用夸张营销文案。
4. Artifact 标题应短、清晰、中文优先；必要英文可括注。
5. 执行计划中优先使用“检查、实现、验证、回滚”这类动作词。
6. 未覆盖英文短语需写入 `reports/untranslated.json`，后续按词库补齐。
7. UI 注入仅能运行于本地 DevTools 连接，不能修改 Antigravity 应用包。
